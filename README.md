## About the Project

This is the frontend application for RoboRally, a strategic board game where players program robots to navigate through a hazardous factory floor. Built with Next.js 15 and TypeScript, the application features a modern cyberpunk-themed UI with real-time multiplayer gameplay using SignalR for WebSocket communication.

### Key Technologies

- **Next.js 15** with App Router and Server Components
- **TypeScript** for type-safe development
- **Tailwind CSS v4** for responsive styling
- **Redux Toolkit** with RTK Query for state management
- **SignalR** for real-time multiplayer communication
- **shadcn/ui** components for the UI library
